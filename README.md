# Weather Analytics Dashboard

End-to-end weather analytics platform built using Open-Meteo, MongoDB, ClickHouse, Redis, and Flask.

## Architecture

Open-Meteo API
→ MongoDB
→ ClickHouse
→ Redis Cache
→ Flask API
→ Dashboard

## Technologies

- Python
- MongoDB
- ClickHouse
- Redis
- Flask
- REST API

## Features

- Automated weather data ingestion
- ETL pipeline implementation
- Data aggregation and caching
- REST API development
- Interactive analytics dashboard

## Components

- api.py – Open-Meteo data ingestion
- mongo_to_clickhouse_enriched.py – ETL pipeline
- clickhouse_aggregation.py – Aggregation layer
- dashboard_app.py – Dashboard API
- templates/index.html – Dashboard UI
